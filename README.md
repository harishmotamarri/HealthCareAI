# MediMind AI (AICARE)

MediMind AI is a comprehensive frontend web application acting as a Personal AI Doctor UI dashboard. It transforms complex medical documents like prescriptions, lab reports, and symptoms into simple, actionable health insights.

## Features

- **PRO MAX UI Design**: Features deep glassmorphic layers, animated background orbs, and rich hover micro-interactions powered strictly by custom Native CSS.
- **Responsive Layout**: Optimized styling mapping fluidly to both mobile and large-screen display configurations.
- **Clear File Architecture**: Clean separation between standard views (`index.html`, `dashboard.html`), styles, and script bundles.

## Setup Instructions

1. **Clone the repository:**
   ```bash
   git clone [your-repo-link]
   cd AICARE
   ```
2. **Access the Application**:
   Simply open `index.html` inside your browser to start.

## Folder Structure

```text
/assets/css/
  dashboard.css - Dedicated styles for the dashboard interface.
  landing.css - Dedicated styles for the marketing pages.
  shared.css - Contains generic style variables and fallbacks.
/assets/js/
  dashboard.js - Interactive view management logic.
  landing.js - Logic for the dynamic landing page demo features.
index.html - The main marketing landing page.
dashboard.html - The main app dashboard page.
```
